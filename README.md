# Mweru Handy Finds Connect

An open-source platform connecting employers and skilled workers in Gweru.

## Project info

- Repository: https://github.com/Eugene-maposa/gweru-handy-finds-connect.git
- Tech stack: Vite, React, TypeScript, shadcn-ui, Tailwind CSS, Supabase

## Getting started locally

1. Clone the repo  
   `git clone https://github.com/Eugene-maposa/gweru-handy-finds-connect.git`
2. Install dependencies  
   `npm install`
3. Run the dev server  
   `npm run dev`
4. Open the app at the URL shown in the terminal (default: `http://localhost:5173`).

## Deploying

Deploy to your preferred host (e.g., Vercel, Netlify, or a static host).  
Build the production bundle with `npm run build` and serve the `dist` output.

### Environment

- `APP_BASE_URL`: set to your deployed app origin (used in welcome emails), e.g. `https://your-domain.com`
